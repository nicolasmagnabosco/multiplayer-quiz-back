# Multiplayer Quiz - Backend ⚙️

The server-side API and real-time engine for the multiplayer quiz, built with **Node.js**, **Express**, and **Socket.io**.

## 🚀 Setup Instructions

1. **Install Dependencies:**

   npm install

2. **Configure Environment Variables:**

Change the name of the ".env.example" file to ".env" and add your variables following the examples.

(Note: You need a network url to access the website from anywhere that isn't the machine where the app is running, for example, a phone on the same WI-FI network).

3.  **Run Development Server:**

npm run dev
